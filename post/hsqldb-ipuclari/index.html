HSQLDB İpuçları

  1. HSQLDB’yi sunucu kipinde çalıştırmak için : java -cp ./hsqldb.jar org.hsqldb.Server -database.0 file:veritabanı dosya adı -dbname.0 veritabanı alias adı komutu girilmlidi. Bu komut hsqldb.jar dosyasının olduğu klasör içinde verilmiştir.
    1. HSQLDB’ye dosya kipinde bağlanırken veritabanı dosya yolunu jar dosyanıza göreceli verebilirsiniz. Jar dosyanız ile veritabaı dosyanız aynı klasörde bulunuyorlarsa veritabanı bağlantı Stringiniz : "jdbc:hsqldb:file:./veritabanı dosya adı" şeklinde olmalıdır.
